Під темою “Інтелектуальна система автоматизованого підбору персоналу з аналізом компетенцій” мається на увазі веб сервіс, який допомагає роботодавцям економити час на підборі кандидатів. Проект буде реалізований на React + Nest.js. Інтелектуальна частина - аналіз резюме, яке завантажується кандидатом у форматі pdf, або docx через OpenAi API, аналізуються ключові навички та досвід роботи з резюме, порівнюється з вимогами вакансії, потім заявки кандидатів розділяються за пріоритетом по відповідності вимогам. Є два типи акаунтів, роботодавці та кандидати. Роботодавці створюють вакансії, можуть додавати анкети, або тестування для кандидатів. Кандидати можуть створити власний профіль, завантажити резюме, пройти тестування чи анкету, залишити контактні дані. Кожна заявка отримує статус, наприклад “Розглядається”, “Відхилено”, “Прийнято”, щоб швидше відсортувати кандидатів. Також передбачається чат між кандидатом та роботодавцем під вакансією, який дозволяє обговорювати вакансію або ставити уточнюючі питання. Мета проекту - це створення зручної системи для підбору персоналу, яка дозволяє економити час на оцінці заявок кандидатів через ранжування заявок за відповідністю вимогам вакансії. Проект було обрано, бо мені згодом потрібно буде зробити інший проект, опосередковано пов’язаний з підбором кандидатів. Даний проект дозволить розібратися в темі й отримати функціонал, який можливо буде реалізувати в інших схожих проектах.
